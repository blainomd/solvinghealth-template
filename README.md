# SolvingHealth Condition Template

A forkable Next.js template for physician vibe coders. Create a health condition site in minutes.

## Three steps

1. **Fork this repo**
2. **Edit `site.config.ts`** -- name, colors, risk factors, warning signs, connector key
3. **Deploy to Vercel** -- connect your repo, done

## What you get

- Professional health condition landing page
- Sage AI chat widget + Gemini voice
- Claude MCP connector snippet (copy-paste install)
- Email capture
- Ecosystem links to SurgeonValue + co-op.care
- Mobile-responsive, accessible, fast
- Vercel Analytics built in

## Stack

- Next.js 16.2.2
- React 19
- Tailwind CSS 4
- TypeScript

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## One config file

Everything on the site reads from `site.config.ts`. Change the name, colors, sections, warning signs, and connector key. That is the entire customization surface.
